Змінні

Оголошення змінних 

                        var currentHour = 17;
Виведення значень змінних на екран 

                        alert(currentHour);

Типи змінних

  • string (рядок): "Name", 'hello'
  • number (числовий): 27, 98.33, спеціальні значення: NaN, Infinity
  • boolean (логічний): true, false
  • null
  • undefined
  • object(об'єкт): {"name": "Alex", "age": "15"}, [1, 2, 3]

Арифметичні оператори

  • +, -, *, /, %

Оператори порівняння

  • >, <, >=, <=, ==, !=
  • && (і)
  • || (або)

Умовні оператори


                        if (condition1) {
                            statement1;
                        } else if (condition2)  {
                            statement2;
                        } else {
                            statement3;
                        }

Функції


                        function (a, b) {
                            console.log(a);
                            return a + b;
                        }

Програмуємо (Let's code)

Створюємо функцію (калькулятор), яка приймає 3 аргументи: число, число і знак операції ("+", "-", "*", "/") і виводить за допомогою alert() результат.
